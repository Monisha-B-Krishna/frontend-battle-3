# Veridian — Verified Data Intelligence Platform

A premium, high-converting SaaS landing page built for **Frontend Battle 3.0** by IIT Bhubaneswar.

## 🔗 Links
- **Live Demo:** https://monisha-b-krishna.github.io/frontend-battle-3/
- **GitHub Repo:** https://github.com/Monisha-B-Krishna/frontend-battle-3

## 🛠 Tech Stack
- Vanilla HTML5 + CSS3 + JavaScript
- Bootstrap 5.3
- Google Fonts (JetBrains Mono + Inter)
- Native Web Animations API (WAAPI)

## ✅ Features Built
### Feature 1 — Matrix-Driven Pricing & Currency Switcher
- Dynamic multi-dimensional pricing matrix in JavaScript
- Toggle between Monthly / Annual (20% discount)
- Three currencies: USD, INR, EUR with regional tariff multipliers
- State-isolated updates via WAAPI — zero global re-renders

### Feature 2 — Bento-to-Accordion with State Persistence
- Desktop: CSS Grid Bento layout with product mockups
- Mobile: Touch-optimized Accordion
- Active index context transfers on window resize crossing 768px breakpoint

## 🎨 Design System
- **Colors:** Arctic Powder, Mystic Mint, Forsythia, Deep Saffron, Nocturnal Expedition, Oceanic Noir
- **Fonts:** JetBrains Mono (headings) + Inter (body)
- **Animations:** 175ms ease-out micro, 350ms ease-in-out layout

## 📄 Sections
1. Hero — animated entrance sequence with floating UI cards
2. Features — Bento Grid / Accordion with product mockups per card
3. Pricing — Dynamic matrix with currency switcher
4. Social Proof — Interactive testimonial switcher (5 companies × 3 reviews)
5. FAQ — Two-column accordion
6. Footer — Full sitemap

## ⚡ Performance
- Page loader completes within 450ms
- Hardware-accelerated CSS transitions only
- No external animation libraries
- Intersection Observer for scroll animations
